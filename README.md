# CaminhoFit

CaminhoFit is a web application focused on helping people discover places to practice physical activities in their city through an interactive map.

The project is currently in development and aims to evolve into a fitness platform that connects people, locations, and activities in a simple and visual way.

---

## Overview

The idea behind CaminhoFit is to make it easier to find places such as:

- Parks for walking or running  
- Outdoor workout areas  
- Gyms and training centers  
- Public spaces for physical activities  

Instead of searching in multiple places, users can explore everything visually on a map.

---

## Current Features (MVP)

- Interactive map with location markers  
- Sidebar with a list of locations  
- Basic filtering by activity type  
- Map popups with location information and images  
- Responsive layout (mobile-first)  

---

## Tech Stack

- Next.js  
- React  
- TypeScript  
- Tailwind CSS  
- Leaflet (OpenStreetMap)  
- Supabase (PostgreSQL + Storage)  

---

## Project Status

This project is currently under active development.

The current version represents an MVP focused on validating the core idea and user experience.

---

## Future Vision

CaminhoFit is being designed to evolve into a more complete platform, with features such as:

- User accounts  
- Activity tracking  
- Social features (posts and interactions)  
- Reviews and ratings  
- Event discovery  
- Expansion to multiple cities
